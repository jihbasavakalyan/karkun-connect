# Assets

Static media files used across the application.

Store images, icons, fonts, and other binary assets here. Import assets directly in components or reference them via Vite's asset handling.

**Guidelines:**

- Use descriptive, kebab-case filenames (e.g., `logo-primary.svg`)
- Prefer SVG for icons and scalable graphics
- Organize into subfolders (e.g., `images/`, `icons/`, `fonts/`) as the asset library grows
