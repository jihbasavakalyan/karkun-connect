# UI/UX

Design system, wireframes, and UX guidelines for **karkun-connect**.

Use this folder to document:

- Wireframes and mockups
- Design tokens (colors, typography, spacing)
- Component usage guidelines
- Accessibility standards (WCAG compliance targets)
- User flows and interaction patterns

Design artifacts here should align with the PRD and inform the `src/components/` and `src/styles/` implementation.
